# Trigger tests

| Skill | Positive prompts | Negative prompts |
|---|---|---|
| second-brain-operations | “Quero criar meu Segundo Cérebro”; “Conecte este vault”; “Rode o doctor” | “Qual é a capital da França?”; “Busque esta palavra numa nota” |
| pull-brain | “Carrega o cérebro”; “O que sabemos sobre retenção?”; “Continue o projeto” | “Salve esta decisão”; “Crie um vault” |
| push-brain | “Salva isso no cérebro”; “Fecha sessão”; “Registre esta decisão” | “Leia sem alterar”; “Mostre o status do índice” |
| brain-search | “Busque no cérebro por autenticação”; “Reconstrua o índice” | “Mova esta nota”; “Publique conteúdo restrito” |

A positive trigger must identify the requested workflow. A negative trigger must remain outside the skill or route to the more specific companion skill.
