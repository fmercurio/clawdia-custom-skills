# Generic knowledge contracts

- **Canonical over conversational:** durable notes outrank session recall.
- **Intent before folder:** retrieve by project/decision/person/concept; PARA is a physical fallback.
- **Brain Delta:** save only decisions, state changes, reusable lessons, and explicit gaps.
- **Promotion:** a raw capture becomes canonical only after destination, provenance, sensitivity, and ownership are known.
- **Sensitivity:** public, internal, restricted. Restricted search and publication are denied by default.
- **Separation:** vault is knowledge; Hermes home is runtime; backup storage is neither.
- **Progressive capability:** FTS5 first; optional features activate only when configured and detected.
